# XML Features

## Named Actors
If you want an actor to be stored in the pseudo-global metatable.
```xml
<Layer Name = "obj" />
```

```xml
<Layer Name = "pool[1]" />
<Layer Name = "pool[2]" />
<Layer Name = "pool[3]" />
```

```xml
<Layer Name = "lots.of.tables.inside.eachother['with']['lua' .. '_' .. 'syntax']" />
```
